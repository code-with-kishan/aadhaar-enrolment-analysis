# Aadhaar Enrolment & Update Trend Analysis

## Objective
Analyze UIDAI Aadhaar enrolment, demographic updates, and biometric updates
to uncover societal trends, system stress points, and predictive insights.

## Repository Structure
- src/: Core data processing and modeling code
- notebooks/: Analysis and visualization notebooks
- data/: Raw UIDAI datasets (not included in repo)

## Reproducibility
All results in the report can be reproduced using the code in this repository.
